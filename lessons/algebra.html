Linear equations

Solving ax + b = 0 leads to x = -b/a (for a ≠ 0). Example:

Solve 3x - 6 = 0 → x = 2.

Quadratic equations

For ax² + bx + c = 0 use the discriminant Δ = b² - 4ac.

If Δ > 0 two real roots: x = (-b ± √Δ)/(2a).

Example: x² - 5x + 6 = 0 → Δ = 25 - 24 = 1 → roots 2 and 3.

Polynomials — quick tip

Factor by grouping when possible; synthetic division helps with rational roots.
